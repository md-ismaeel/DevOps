What is Cloud Computing?
Cloud Computing is the delivery of computing services—
such as servers, storage, networking, databases, software,
and analytics—over the internet (“the cloud”) on a pay-as-you-go model.

Instead of buying physical hardware, companies use cloud
providers to host and run their applications.

------------------------------------------------------------

Why Cloud Computing?
- No need to maintain physical servers
- Pay only for what you use
- Highly scalable and available
- Fast deployment of applications
- Access from anywhere
- Reduced operational cost

------------------------------------------------------------

Service Models:
1. IaaS – Infrastructure as a Service
   - Provides virtual machines, storage, network
   - Example: AWS EC2, Azure VM, GCP Compute Engine

2. PaaS – Platform as a Service
   - Provides runtime, managed platforms, databases
   - Example: Heroku, AWS Elastic Beanstalk, Google App Engine

3. SaaS – Software as a Service
   - Ready-to-use applications
   - Example: Gmail, Netflix, Salesforce
