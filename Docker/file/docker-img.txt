Building own docker images

- docker build -t my-image .                # build image from current directory
- docker build -t my-image . -f Dockerfile  # build image from Dockerfile
- ADD . or COPY .                           # add current directory to image
- docker run my-image                      # run image
- docker run -p 80:80 my-image             # run image and map port 80 to host port 80
- docker rmi my-image                      # remove image
